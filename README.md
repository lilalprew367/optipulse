## OptiPulse

**AI-Powered Daily Options & Stock Trading Intelligence Platform**

OptiPulse surfaces the **best high-conviction trades** every day by aggregating unusual options flow, Quiver Quant political/smart money signals, TradingView ideas, social sentiment, and more.

### Key Features
- Live Unusual Whales Options Flow
- Quiver Quant + Political Trade Signals
- AI-Powered Trade Thesis (Grok integration coming)
- Performance Tracking & Backtesting
- Real-time Alerts & Risk/Reward Calculator
- Dashboard with SignalFeed + LiveThesis

### Tech Stack
- React + Vite + Tailwind + shadcn/ui
- Base44 backend
- Tanstack Query
- Unusual Whales API, Quiver Quant API, TradingView

### Setup
1. Clone the repo
2. `npm install`
3. Add API keys in Base44 settings or .env
4. `npm run dev`

Built to help you find edge in options trading.
